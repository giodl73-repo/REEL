# Pulse 39: Artifact platform duration verification

## Outcome

Made artifact verification reject platform-level video duration drift.

## Changes

- `artifact-check` now verifies each platform's shot-card video duration against
  the source export duration.
- `artifact-check` now verifies each platform's full work-preview duration
  against the same source export duration.
- Batch and review-all verification paths inherit the stricter duration checks.
- README notes platform video duration coverage.
