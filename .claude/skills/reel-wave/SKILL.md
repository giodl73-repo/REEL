# REEL Wave

Use this skill to plan a multi-pulse REEL wave.

## Workflow

1. Read `CLAUDE.md`, `PRODUCT_PLAN.md`, and `context/waves/PHASES.md`.
2. Identify the active wave or create a dated wave under `context/waves/`.
3. Keep the wave focused on one production capability: format grammar, manifest,
   renderer research, review panel, or a specific video package.
4. Name 3-5 pulses, each with validation commands.
5. Record explicit non-goals, especially renderer/provider choices not yet
   researched.

## Output

Update the wave `WAVE.md` and create the first pulse file if execution starts.
