# REEL Research

Use this skill before choosing a renderer, video model, manifest schema, export
target, or broad production workflow.

## Workflow

1. State the decision and acceptance criteria.
2. Compare options with citations or local evidence.
3. Include licensing, cost, automation, quality, reproducibility, and platform
   constraints.
4. Recommend one path and one fallback.
5. Record the result in the active wave before implementation.

## Common research topics

- FFmpeg vs Remotion vs Blender vs browser capture for deterministic rendering.
- AI-video provider capabilities and usage constraints.
- Caption, audio, and aspect-ratio export standards.
- Storage policy for generated renders and preview fixtures.
